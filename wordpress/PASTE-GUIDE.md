# 固定ページに本文を貼り付ける手順

サーバー操作なしで、WordPress管理画面の固定ページに直接貼り付ける方式の手順です。

## 1. HTML本文を入手

[wordpress/page-body.html](page-body.html) を開いて、**Raw** ボタンから全文をコピーします。
- 直接URL: https://raw.githubusercontent.com/onotaisei-create/mycareer-subsidy-plan-mock/main/wordpress/page-body.html

## 2. 固定ページに貼り付け

1. WP管理画面 → 固定ページ → 新規追加
2. **タイトル**: 助成金活用プラン
3. エディタ右上のタブを「**ビジュアル**」から「**テキスト**」に切り替え（重要：HTMLモード）
4. テキストエリアにコピーしたHTML全文を貼り付け
5. **以降ビジュアルタブには戻さない**（戻すとWordPressがCSSやタグを書き換える可能性あり）

## 3. パーマリンク設定

1. タイトルの下に表示される「パーマリンク」を編集
2. スラッグを `subsidy-plan` に変更
3. 確定URL: `https://my-career.co.jp/subsidy-plan/`

## 4. ページ属性

- テンプレート: **デフォルトテンプレート** のまま（ファイル方式ではないため）
- 親: なし

## 5. 公開・確認

1. 「公開」ボタンをクリック
2. 「ページを表示」で実際の表示を確認
3. ヘッダー / ハンバーガー / フッターは既存テーマのものが自動で表示されます

## トラブルシューティング

### CSSが効かない・崩れている
- 「ビジュアル」タブに切り替えてしまった可能性 → 取り消すかバックアップから再貼り付け
- 一部テーマは `<style>` タグを除去します。その場合は外部CSSファイル化が必要なので連絡してください

### お問い合わせボタンのリンク先を変えたい
- 貼り付け本文の `href="/contact/"` を実際のお問い合わせページURL（例: `/contact_kigyo/`）に書き換え

### タイトル「助成金活用プラン」がページ上部にも表示されてしまう
- テーマが固定ページのタイトルを自動表示している場合
- 対応案A: そのまま（SEO的にも問題なし）
- 対応案B: 該当ページのみCSSで非表示にする（依頼があれば対応）

### モバイル表示が崩れる
- スマホで再確認してください。テーマ既存のブレークポイント（959px / 559px）に合わせています
- 崩れる場合はスクショ送ってください

## 編集する場合

このHTMLを再編集するときは、必ず**テキストタブで開いてテキストタブで編集・保存**。ビジュアルタブを経由するとCSSとレイアウトが壊れます。
