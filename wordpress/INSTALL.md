# WordPress 実装手順

`mycareer_2023` テーマに「助成金活用プラン」ページを追加する手順です。

## 1. テンプレートファイルをテーマに配置

`page-subsidy-plan.php` を WordPress サーバー上の以下に配置します:

```
/wp/wp-content/themes/mycareer_2023/page-subsidy-plan.php
```

配置方法（いずれか）:
- **お名前.com Web Filer**（推奨／ブラウザだけで完結） → [ONAMAE-UPLOAD.md](ONAMAE-UPLOAD.md) 参照
- **FTP / SFTP**（Cyberduck / FileZilla 等） → [ONAMAE-UPLOAD.md](ONAMAE-UPLOAD.md) 参照
- **WordPress 管理画面**：外観 → テーマファイルエディター → 既存ファイル編集のみで新規作成不可なので使えない

## 2. WordPress 管理画面で固定ページを作成

1. 固定ページ → 新規追加
2. **タイトル**：助成金活用プラン
3. **URLスラッグ**：`subsidy-plan`（パーマリンクが `/subsidy-plan/` になる）
4. **本文**：空のままで OK（テンプレート側で全コンテンツを出力）
5. 右サイドバー「ページ属性」→ **テンプレート** で「**助成金活用プラン**」を選択
6. 公開

## 3. 動作確認

- 公開URL: `https://my-career.co.jp/subsidy-plan/`
- ヘッダー / ハンバーガーメニュー / フッターは既存テーマのものが自動で適用されます
- スマホ表示も既存テーマのレスポンシブ仕様（959px / 559px ブレークポイント）に合わせています

## 4. 編集方法

### コンテンツの修正
`page-subsidy-plan.php` 内のHTMLを直接編集してアップロード。

### お問い合わせリンク先の変更
ファイル末尾の CTA ボタンは `home_url( '/contact/' )` を指しています。別URLにしたい場合はそこを変更。

### カラー・装飾の変更
ファイル冒頭の `<style>` ブロック内で完結。すべてのクラスに `sp-` プレフィックスを付けているのでテーマ側CSSと干渉しません（`hd-h3` のみテーマの既存クラスを利用）。

## 5. 既存ナビゲーションへの追加

グローバルナビに「助成金プラン」を追加したい場合:
- 管理画面 → 外観 → メニュー → 該当メニューに固定ページ「助成金活用プラン」を追加

ハンバーガーナビ（モバイル）も同じメニューに紐づくはずなので、両方反映されます。
